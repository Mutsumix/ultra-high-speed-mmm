# Ultra High Speed MMM

特定の YouTube 動画の再生速度を簡単に変更できるモバイルアプリ

## 機能

- YouTube 動画の再生
- 再生速度の変更（0.5x, 1x, 2x）
- 最初から再生/続きから再生
- ループ再生機能
- 画面回転対応

## 使用技術

- React Native (Expo)
- YouTube iframe Player API
- @expo/vector-icons

## 開発環境のセットアップ

```bash
git clone https://github.com/your-username/ultra-high-speed-mmm.git
cd ultra-high-speed-mmm
npm install
npx expo start
```
